WebExtension Playground

This Glitch allows you to build and test a browser extension in the browser.

  1. Click the Fish in the top right
  2. Choose "Remix on Glitch" to copy this project
  3. Learn about Browser Extensions on MDN
  4. Edit the files in the /extension directory in the Glitch editor
  5. View this page by clicking "Show [Live]" above
  6. Click the "Install!" button below
